Respaldo automático (recomendado)

Esta página sirve para respaldar cuando quieras, pero lo ideal es que se haga solo. Para dejarlo programado:
  1. Entra a la consola de Google Cloud del proyecto sigpaf-cre.
  2. Activa la facturación del proyecto (la exportación programada la requiere; el costo para este volumen es mínimo).
  3. Crea un bucket de Cloud Storage, por ejemplo sigpaf-respaldos.
  4. En Firestore → Importar/Exportar, programa una exportación diaria o semanal hacia ese bucket.
  5. Define cuánto tiempo se conservan las copias (por ejemplo, 90 días).
Mientras eso no esté configurado, descarga un respaldo manual al menos una vez por semana y al cerrar cada mes.
El respaldo refleja los datos al momento de descargarlo. No incluye las cuentas de acceso (esas las administra Firebase).
